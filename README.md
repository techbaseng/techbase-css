## About This Course

CSS (Cascading Style Sheets) is what makes the web beautiful. While HTML gives a page its structure, CSS controls how everything looks — colors, fonts, spacing, layout, and animations.

In this course, you will learn how to take a plain HTML page and transform it into a polished, visually appealing, and fully responsive website.

**Recommended: Complete the HTML Fundamentals course before starting this one.**

---

## What You Will Learn

By the end of this course you will be able to:

- Connect a CSS file to an HTML page
- Style text, colors, backgrounds, and borders
- Use the CSS Box Model to control spacing and layout
- Build flexible layouts using Flexbox
- Create responsive pages that work on phones and desktops
- Add transitions and simple animations to elements

## How to Use This Course

1. **Start here** — read this page to understand the full picture
2. **Open Lesson 01** — click the link in the table below
3. **Read the lesson** — each file explains the concept with examples and code
4. **Try the exercises** — every lesson ends with hands-on practice tasks
5. **Move forward** — go to the next lesson whenever you are ready

> 💡 **Tip:** Always have your HTML file open alongside your CSS file. Every CSS change you make should be immediately visible when you refresh your browser.
